# fizzbuzz-app
An app based on the Fizzbuzz programming exercise

### Getting Started
If you just want to play the Fizzbuzz game yourself, you'll need to download the fizzbuzz.apk file onto your phone and then install the app. If you want to look at source code, you can simply copy paste all the source files straight into Android Studio.

### Installation

Viewing Code:
You'll need to install Android Studio : https://developer.android.com/studio

Viewing App:
Can be done either through Android Studio on a virtual device, or with your own android phone through installation of provided .apk file.
